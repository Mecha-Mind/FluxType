# FluxType — لعبة الطباعة السريعة

مشروع واجهة أمامية بسيطة (HTML + CSS + JavaScript فقط، بدون إطار عمل أو بنّاء). الهدف هو تمرين الطباعة السريعة والدقة عبر كلمات إنجليزية تظهر واحدة تلو الأخرى مع عدّ زمني لكل مستوى صعوبة.

---
---

## الخلاصة

المشروع يشرح تدفقاً منظّماً: **تحميل → قائمة → لعبة (شرح اختياري/إلزامي لأول مرة) → جولة لعب (بدون زر بداية ظاهر) → شاشة نهاية ملء الشاشة مع أعلى نتيجة**. فهم الملفات يبدأ من `typinggame.html` ثم `type.css` ثم `type.js` مع `design-tokens.css`.
